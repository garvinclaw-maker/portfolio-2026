AI Audio Renamer.

Semantic file organization for recording artists and music producers.

Problem

Music producers often deal with thousands of audio files with cryptic names like "REC001.wav." Manually renaming these to reflect the vibe or content of the audio is a time-consuming manual process that halts creative flow.

Solution

The AI Audio Renamer uses an autonomous agentic workflow to scan audio directories. It generates theme-based, evocative names for audio files based on producer preferences, using LLMs (OpenAI/Ollama) to ensure semantic relevance and consistency.

Implementation

The engine is built to interface with local file systems, allowing producers to "refactor" their entire sample library with a single command while maintaining the original metadata.

Stack: Python, OpenClaw, Ollama, OpenAI API
