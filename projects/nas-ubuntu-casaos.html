Infrastructure

Laptop NAS (CasaOS).

Repurposing old hardware into a high-performance local cloud and media server.

Concept

Instead of purchasing dedicated server hardware, this project focuses on sustainable tech by converting an end-of-life laptop into a fully functional Network Attached Storage (NAS) and application server.

Technical Stack

  • OS: Ubuntu Server (Headless)
  • Interface: CasaOS for Docker container management.
  • Services: Plex Media Server, Pi-hole (network-wide ad blocking), Nextcloud, and local AI model hosting.

Key Optimization

The system was optimized for low-power consumption and heat management, ensuring stable 24/7 uptime. It serves as the primary data backup for all my architectural and product design work.

Focus: DevOps, Infrastructure, Sustainability

Tools: Docker, SSH, Linux Administration
