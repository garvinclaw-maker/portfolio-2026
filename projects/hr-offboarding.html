HR Offboarding.

Streamlining internal operations using agentic presentation generation.

Objective

The transition period during employee offboarding is often chaotic, with critical knowledge lost in the shuffle. This project focuses on automating the creation of transition decks and documentation.

Automation Workflow

  • Data Aggregation: Using OpenClaw to scan a departing employee's project boards, tasks, and documentation folders.
  • Synthesis: LLMs categorize the data into 'Critical Actions,' 'Ongoing Projects,' and 'System Access.'
  • Generation: Automatically generating a professional PowerPoint (PPT) presentation that summarizes the status for the incoming replacement.

Value Proposition

By automating the manual documentation phase, the engine ensures 100% knowledge retention and reduces the overhead for both the HR team and the departing staff member.

Stack: OpenClaw, Python (python-pptx), LLM Orchestration
